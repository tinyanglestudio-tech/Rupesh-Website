## Deploy in 3 steps

### Step 1 — Upload to GitHub
1. Go to github.com → New repository
2. Name it: `rupesh-shinde`
3. Set to **Public**
4. Click **Create repository**
5. Click **Add file → Upload files**
6. Drag all files from this folder (`index.html`, `vercel.json`, `README.md`)
7. Click **Commit changes**

### Step 2 — Deploy on Vercel
1. Go to vercel.com → Sign up with GitHub
2. Click **Add New → Project**
3. Select `rupesh-shinde` → Click **Import**
4. Click **Deploy** — live in ~1 minute

### Step 3 — Your site is live at
```
https://rupesh-shinde.vercel.app
```

## Add a custom domain (optional)
1. Buy `rupeshshinde.com` from Namecheap (~$10/yr)
2. Vercel Dashboard → Settings → Domains → Add domain
3. Update DNS records as shown → live in minutes
